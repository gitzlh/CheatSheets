# Python 正则表达式

## 正则表达式基础
![alt text](regex.png "Logo Title Text 1")
#### 一般字符
- `.` 匹配任意字符（\n 除外）
- `\` 转义字符

#### 用[]精确匹配
- `[0-9a-zA-Z\_]` 匹配一个数字、字母或者下划线 
- `[0-9a-zA-Z\_]+` 匹配至少由一个数字、字母或者下划线组成的字符,如`0z_`
- `[0-9a-zA-Z\_]{2,5}` 匹配至少由2-5个数字、字母或者下划线组成的字符,如`0z__`
- 


#### 数量词
- `*` 任意个字符（包括0个)
- `+` 至少一个字符
- `?` 0个或1个字符
- `{n}` n个字符
- `{n,m}` 表示n-m个字符
#### 字符集
- `\d` 数字[0-9]
- `\w` 单词+数字 [a-zA-Z0-9]
- `\s` 空白字符
- `大写匹配其否定,e.g., `\D` 匹配非数字
